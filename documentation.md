# 신재인랭 Docs

## 자료형
### 정수
`ㅇㄴ` 와 `ㅇㅂㄷ` 로 표현합니다. `ㅇㅂㄷ` 앞에 오는 `ㅇㄴ` 의 개수만큼의 값을 가집니다.
```
ㅇㅂㄷ => 0
ㅇㄴㅇㅂㄷ => 1
ㅇㄴㅇㄴㅇㄴㅇㄴㅇㄴㅇㅂㄷ => 5
```

## 변수
신재인랭에는 274개의 정수 변수가 있으며, 각각 1,2,3..274의 인덱스를 가집니다.

변수는 `어` 를 이용해 접근할 수 있습니다. `어` 의 개수를 인덱스로 가지는 변수에 접근합니다.
```
어 => 1번 변수
어어 => 2번 변수
어어어어어 => 5번 변수
```

## 연산자
### 엄
값을 대입합니다.
```
어 엄 ㅇㄴㅇㅂㄷ => 1번 변수에 1 대입
어어 엄 ㅇㄴㅇㄴㅇㄴㅇㄴㅇㄴㅇㅂㄷ => 2번 변수에 5 대입
```
### 히히
두 정수를 더하거나, 플러스 기호의 역할을 합니다.
```
ㅇㄴ ㅇㅂㄷ 히히 ㅇㄴ ㅇㅂㄷ => 1+1 = 2
ㅇㄴ ㅇㄴ ㅇㄴ ㅇㅂㄷ 히히 ㅇㄴ ㅇㅂㄷ => 3+1 = 4
```
### 손가락
두 정수를 빼거나, 마이너스 기호의 역할을 합니다.
```
ㅇㄴ ㅇㄴ ㅇㄴ ㅇㅂㄷ 손가락 ㅇㄴ ㅇㅂㄷ => 3-1 = 2
손가락 ㅇㄴ ㅇㅂㄷ => -1
```
### ㅅㅂ?
두 정수를 곱합니다.
```
ㅇㄴ ㅇㄴ ㅇㅂㄷ ㅅㅂ? ㅇㄴ ㅇㄴ ㅇㅂㄷ => 2*2 = 4
```

### 연산자 우선순위

| 우선순위 | 연산자 |
| :---: | --- |
| 1 | ㅅㅂ? |
| 2 | 히히, 손가락 |
| 3 | 엄 |

## 구분자
### ? 와 ㅇㅇ
명령어 앞에는 `?` 를, 뒤에는 `ㅇㅇ` 을 씁니다.

## 명령어
### 졌진 않았잖아
한 개의 문자를 입력받고, 그 문자의 유니코드 코드값을 반환합니다.
```
어 엄 ? 졌진 않았잖아 ㅇㅇ => 입력값을 1번 변수에 대입
```
### 나는 솔직히 잘생김
정수 하나를 인자로 받아, 그 값을 코드로 가지는 유니코드 문자를 출력합니다.
```
어 ㅇㄴㅇㄴㅇㄴㅇㄴㅇㄴㅇㅂㄷ => 1번 변수에 5 저장
어어 어 ㅅㅂ? ㅇㄴㅇㄴㅇㅂㄷ ㅅㅂ? 어 히히 어 ㅅㅂ? ㅇㄴ ㅇㄴ ㅇㄴ ㅇㅂㄷ => 2번 변수에 5*2*5 + 5*3 = 65 저장
? 나는 솔직히 잘생김 ㅇㅇ 어어 => 2번 변수의 값(65)를 코드로 가지는 유니코드 문자('A') 출력
```
