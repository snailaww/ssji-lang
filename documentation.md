# 신재인랭 Docs

## 자료형
### 정수
`ㅇㄴ`와 `ㅇㅂㄷ`를 이용해 표현합니다. `ㅇㅂㄷ` 앞에 오는 `ㅇㄴ`의 개수만큼의 값을 가집니다.

```
ㅇㅂㄷ => 0
ㅇㄴ ㅇㅂㄷ => 1
ㅇㄴ ㅇㄴ ㅇㄴ ㅇㄴ ㅇㄴ ㅇㅂㄷ => 5
```

## 변수
총 274개의 변수가 있으며, 1..274의 인덱스를 가지고 있습니다. **인덱스가 1부터 시작함에 유의하세요.**

변수는 `어` 의 개수로 인덱스를 나타냅니다.
```
어 => 1번 변수
어어어 => 3번 변수
```

## 연산자
### 대입 연산자
`엄` 으로 변수에 값을 대입할 수 있습니다.
```
어어어 엄 ㅇㄴ ㅇㅂㄷ => 3번 변수에 1 대입
```

### 사칙연산자
- 덧셈 : `히히`
- 뺄셈 : `손가락`
- 곱셈 : `ㅅㅂ?`
- 나눗셈 (몫) :
- 나눗셈 (나머지) :

## 명령어
모든 명령어 뒤에는 명령어가 끝났음을 알려주는 `ㅔ?` 가 있어야 합니다.
### 출력
`솔직히 나는 잘생김 ㅇㅇ` 뒤에 오는 정수/변수를 코드로 가지는 유니코드 문자를 출력합니다. 반환값은 항상 `ㅇㅂㄷ` 입니다.
```
어 엄 ㅇㄴ ㅇㄴ ㅇㄴ ㅇㄴ ㅇㄴ ㅇㅂㄷ => 1번 변수 = 5
어어 엄 어 ㅅㅂ? ㅇㄴ ㅇㄴ ㅇㅂㄷ => 2번 변수 = 2*5 = 10
어어어 엄 어어 ㅅㅂ? 어 히히 어 ㅇㄴ ㅇㄴ ㅇㄴ ㅇㅂㄷ => 3번 변수 = 10*5+5*3 = 65 
솔직히 나는 잘생김 ㅇㅇ ㅔ? 어어 => 3변 변수의 값(65)를 코드로 가지는 유니코드 문자 ('A') 출력
```

### 입력
`졌진 않았잖아`로 입력된 문자의 유니코드 코드를 반환합니다. **입력은 한 문자씩 받습니다.**
